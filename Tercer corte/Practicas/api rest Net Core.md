# API REST

## Node.js

Instalamos Dotnet NET CORE 2.2 [^1]

Se configuro el puerto publico mediante [^2]

## clonamos git

```apache
git clone https://github.com/miguel-rondon/api-rest-cs.git
```

## Ejecutamos servicio

```apache
dotnet run
```
